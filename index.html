HTML & CSS Lesson

HTML

If you utilize the inspector tool (ctrl/cmd shift c) and look under the hood of this webpage, you will find this to be a section tag. This is one of HTML's semantic elmeents. They're much easier to read than a div.

CSS

This particular element is inside of a div tag. These elements are containers for any tag that you'd like and do not require having a heading unlike the section tag. They are the most popular but less human-friendly when it comes to reading code.
